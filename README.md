# Пошук у ширину (Breadth-First Search, BFS)

## 1. Основи графів: вершини, ребра, шляхи

### Граф

Граф — це структура даних, яка складається з:

* **вершин (vertices)** — точки графа;
* **ребер (edges)** — зв'язки між вершинами.

Приклад графа:

```text
A --- B
|     |
C --- D
```

Тут:

* вершини: A, B, C, D;
* ребра: A-B, A-C, B-D, C-D.

### Шлях

Шлях — це послідовність вершин, з'єднаних ребрами.

Наприклад:

```text
A → B → D
```

Довжина шляху дорівнює кількості ребер.

---

## 2. Представлення графів у пам'яті

### Матриця суміжності

Граф можна зберігати у вигляді таблиці.

Приклад:

|   | A | B | C | D |
| - | - | - | - | - |
| A | 0 | 1 | 1 | 0 |
| B | 1 | 0 | 0 | 1 |
| C | 1 | 0 | 0 | 1 |
| D | 0 | 1 | 1 | 0 |

1 означає наявність ребра між вершинами.

### Список суміжності

Для кожної вершини зберігається список її сусідів.

```python
graph = {
    'A': ['B', 'C'],
    'B': ['A', 'D'],
    'C': ['A', 'D'],
    'D': ['B', 'C']
}
```

Цей спосіб використовується найчастіше.

## 3. BFS: алгоритм і його реалізація

### Ідея алгоритму

Пошук у ширину (BFS) обходить граф рівень за рівнем.

Для роботи використовується структура даних **черга (Queue)**.

Послідовність роботи:

1. Почати з початкової вершини.
2. Додати її в чергу.
3. Взяти вершину з черги.
4. Відвідати всіх її сусідів.
5. Невідвіданих сусідів додати в чергу.
6. Повторювати, поки черга не стане порожньою.

### Реалізація на Python

```python
from collections import deque

def bfs(graph, start):
    visited = set()
    queue = deque([start])

    visited.add(start)

    while queue:
        vertex = queue.popleft()
        print(vertex)

        for neighbor in graph[vertex]:
            if neighbor not in visited:
                visited.add(neighbor)
                queue.append(neighbor)

graph = {
    'A': ['B', 'C'],
    'B': ['A', 'D'],
    'C': ['A', 'D'],
    'D': ['B', 'C']
}

bfs(graph, 'A')
```

Результат:

```text
A
B
C
D
```

### Складність алгоритму

* Часова складність: **O(V + E)**
* Просторова складність: **O(V)**

де:

* V — кількість вершин;
* E — кількість ребер.

## 4. Приклади використання BFS у житті

### Навігація

Пошук найкоротшого шляху між точками на карті.

### Соціальні мережі

Пошук друзів друзів та рекомендація знайомств.

### Комп'ютерні мережі

Пошук маршрутів передачі даних.

### Ігри

Пошук найкоротшого маршруту персонажа до цілі.

### Веб-пошук

Обхід сторінок сайту пошуковими роботами.

---

## Висновок

Пошук у ширину (BFS) — це алгоритм обходу графа, який відвідує вершини рівень за рівнем. Він використовує чергу та гарантує знаходження найкоротшого шляху в незважених графах.
